# Superheroes (f)

Deze oefening is het vierde deel van de oefeningenreeks "superheroes". We gaan verder op de oplossing van het vijfde deel (e).

## Opgave

Werk verder op de bestanden uit oefening e. Deze pagina's zouden al reeds gekoppeld moeten zijn met een stylesheet `mainstyle.css`.

### index pagina

Breng de volgende wijzigingen aan op de `index.html` pagina:

- Verplaats het **div** element met de navigatie erin naar boven op de pagina.
- Voeg een **class** attribuut toe op dit **div** element en geef dit de waarde **navigation**.
    - De **border** en **border-radius** properties worden beiden terug op 0 geplaatst.
    - De achtergrondkleur wordt dezelfde kleur als deze op het `body` element.
    - We geven enkel de onderste border een dikte van `2 pixels`, een stijl `dotted` en dezelfde kleur als onze andere **div** borders

- Plaats de **a** elementen in een **unordered list** element, voorzie een **list item** per element.
    - Wijzig de **font-size** van het **ul** element naar `1.5rem`
    - Voorzie een icoontje voor de list-items (Je kan gebruik maken van `listIcon.png` uit de sources map)
    - Plaats dit via css **in 1 regel** aan de buitenzijde van het **ul** element

- Voorzie volgende zaken in je css stylesheet voor alle **a** elementen die zich binnen de navigatie bevinden:
    - Geen lijn meer onder de link/tekst.
    - Een **vette** opmaak van de tekst.
    - Een rode kleur van de tekst.

### footer

Pas de footer op iedere html pagina aan zodat:

- De tekst gecentreerd wordt
- De **border** en **border-radius** properties bijden op 0 staan
- De achtergrondkleur dezelfde kleur als deze op het **body** element wordt
- Enkel de bovenste border krijgt een dikte van `2 pixels`, een stijl `dotted` en dezelfde kleur als onze andere **div** borders

### artikel pagina's

- Plaats de juiste klasse op je bovenste **div** element (onze navigatie).
- Plaats het **a** element in een **ul** element als een **li** element
- Plaats de juiste klasse op de alinea's
- Plaats de juiste klasse op het onderste *8div** element (onze footer).

## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
