# Superheroes (c)

Deze oefening is het derde deel van de oefeningenreeks "superheroes". We gaan verder op de oplossing van het tweede deel (b).

## Opgave

Werk verder op het bestand `index.html` uit oefening b. Deze pagina zou al reeds gekoppeld moeten zijn met een stylesheet `mainstyle.css`.

Voor volgende stappen uit met een editor naar keuze.
- Pas in het eerste artikel, de eerste alinea (**p** element) aan:
    - Leg een nadruk (*emphasis*) op onderstaande stukjes uit de alinea. Gebruik hiervoor het passende html element.
        - slechts enkelen
        - zo goed als vergeten
        - verdienen
- Pas in het tweede artikel het volgende aan:
    - Pas de eerste zin van de eerste alinea (**p** element) aan zodat de eerste zin met een passend element in het **vet** geplaatst wordt.
    - De tweede zin begint op een nieuw regel.
    - Plaats de verwijzingen naar de shows in de tweede alinea in passende elementen. Verwijder nadien de overbodige aanhalingstekens die rondom de oorspronkelijke tekst stonden.
    - Doe hetzelfde voor de shows vermeld in de derde alinea.
- Pas de volgende zaken aan in het derde artikel:
    - De eerste zin van de eerste alinea (**p** element) moet aangepast worden zodat de eerste zin met een passend element in het **vet** geplaatst wordt.
    - De tweede zin begint op een nieuwe regel.
    - Plaats in de laatste alinea een passend element rondom *Tibet in Comics*.
- Pas in het vierde artikel onderstaan aan:
    - Plaats in de laatste alinea een passend element rondom *League of Extraordinary Gentlemen*
- Pas de volgende zaken aan in het vijfde artikel.
    - Plaats de vier superhelden uit dit artikel in een passend element.
- Voorzie een passender element voor onze copyright en brontekst odneraan de pagina. *(Tip: deze mag wat kleiner zijn*)

## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
