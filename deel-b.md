# Superheroes (b)

Deze oefening is het tweede deel van de oefeningenreeks "superheroes". We gaan verder op de oplossing van het eerste deel (a).

## Opgave

Start met je oplossing van deel a van de superheroes oefening en maak een nieuwe folder met de naam `css` aan. Voorzie hierin een bestand `mainstyle.css` waarin we de opmaak van onze pagina gaan beschrijven. Voer vervolgens volgende stappen uit in een editor naar keuze (bijvoorbeeld Visual Studio Code).

- Koppel je css bestand aan de `index.html` pagina
- Voeg de volgende opmaakregels toe aan het **body** element
    - Geef de volgende waarden aan de verschillende **marges** en probeer dit met 1 regel css te doen:
        - **top**: `10px`
        - **right**: `150px`
        - **bottom**: `50px`
        - **left**: `150px`
        - Zet de achtergrondkleur op lichtgeel
    - Zet de tekstkleur van alle **h1** element op rood
    - Voeg de volgende opmaak toe aan de **span** elementen
        - Rode tekstkleur
        - Plaats de tekst cursief (**italic**)
        - Plaats de tekst vet (**bold**)
    - Voeg de volgende opmaak toe aan de **div** elementen
        - Geef de volgende waardes aan de **padding**
            - **top**: `5px`
            - **right**: `10px`
            - **bottom**: `15px`
            - **left**: `20px`
        - Zet de **bottom-margin** op `20px`
        - Voorzie een oranje achtergrond met een transparantie (**transparancy**/**opacity**) van 25%
        - Voeg een border toe van `1px` die een donkerrode kleur heeft
        - Rond de linkerbovenhoek en rechteronderhoek af op `50px`

## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
