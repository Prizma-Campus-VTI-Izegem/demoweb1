# Superheroes (e)

Deze oefening is het vierde deel van de oefeningenreeks "superheroes". We gaan verder op de oplossing van het vierde deel (d).

## Opgave

Werk verder op de bestanden uit oefening d. Deze pagina's zouden al reeds gekoppeld moeten zijn met een stylesheet `mainstyle.css`.

### index pagina

Breng de volgende wijzigingen aan op de `index.html` pagina:

- Vervang het **h1** element door een **p** element. Plaats een **class** attributt met de waarde `logo` op dit nieuwe **p** element.
- Plaats binnenin dit nieuwe **p** element de afbeelding `superheroIncLogo.png`. Je kan deze afbeelding vinden in de `sources` map
- Centreer de afbeelding via css, maak gebruik van de **selector op class niveau** om dit te bekomen
- Zorg voor een passende invulling van de waarde van het **alt** attribuut

- Plaats in het **div** element waar de sectie `Over Ons` in staat net boven het **h2** element een nieuwe afbeelding genaamd `bang.png`. Je vindt deze afbeelding terug in de `sources` map.
- Maak in je `css` een passende klasse aan om deze afbeelding aan de rechterkant te laten "drijven" en pas deze toe op de afbeelding.
- Zorg voor een passende invulling van de waarde van het **alt** attribuut


### Artikel pagina's
Breng volgende wijzigingen aan op de `artikel01.html` pagina:

- plaats de afbeelding `8man.jpg` net boven het **h2** element op deze pagina. Je vindt deze afbeelding in het meegeleverde `afbeeldingen.zip` bestand.
- Maak in je `css` een passende klasse aan om deze afbeelding aan de linkerkant te laten "drijven" en pas deze toe op de afbeelding.
- Plaats het **id** attribuut op het **img** element en geef het de waarde **cover8man**.
- Zorg in je css dat je via een passende selector op het gegeven id volgende zaken toepast:
    - Een breedte van 200px
    - Een margin met volgende eigenschappen:
        - Top: 10px
        - Right: 25px
        - Bottom: 10px
        - Left: 10px
    - Zorg ervoor dat het laatste **p** element niet langer drijft naast de afbeelding

Doe hetzelfde met de afbeelding `greenLama.jpg` op je `artikel02.html` pagina. Zorg er hiervoor dat de afbeelding een hoogte van 275 pixel heeft.
Doe hetzelfde met de afbeelding `stardust.jpg` op je `artikel03.html` pagina. Zorg dat de breedte van deze afbeelding `40` procent van de beschikbare breedte inneemt en dat de margin overal 20px is.

## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
