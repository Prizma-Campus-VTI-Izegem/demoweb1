# Superheroes (a)

Deze oefening is het eerste deel van de oefeningenreeks "superheroes". We starten hier met een leeg project.

Benodigde bestanden:
- [Brontekst_SuperheroInc.txt](Brontekst_SuperheroInc.txt)

## Opgave

Start met een lege folder als uitgangspunt voor je uitwerking.

Voer volgende stappen uit met een editor naar keuze (bijvoorbeeld Visual Studio Code).

- Maak een nieuwe webpagina aan en noem deze `index.html`.
- Geef je webpagina de **titel** `Superhero Inc`.
- Maak gebruik van het tekstdocument [Brontekst_SuperheroInc.txt](sources/Brontekst_SuperheroInc.txt) voor de inhoud van je webpagina.
    - Plaats `Superhero Inc` in een **h1** element.
    - De titels die een **h2** element toegewezen moeten krijgen staan tussen asteristekn (`*`)
        - De andere tekst worden in passen **p** elementen geplaatst
    - De titel `Andere illustere superhelden` wordt in een **h3** element geplaatst.
        - Plaats hierbij de inleidende tekst en de vier superhelden in een afzonderlijk **p** element
- Plaats rondom elk artikel een **div** element. *Een artikel bestaat hierbij uit het **h2**/**h3** element en de bijhorende paragrafen
- Plaats doorheen de tekst op je pagina de woorden `superheld` en `superhelden` in een **span** element. *Ga dit niet één voor één aanpassen, maar zoek hiervoor een passende/effectieve manier)
- Plaats onderaan je pagina een **div** element
    - In dit **div** element plaats je vervolgens een **p** element
    - In dit **p** element voeg je volgende copyright toe `© 2018 Superhero Inc.`

## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
