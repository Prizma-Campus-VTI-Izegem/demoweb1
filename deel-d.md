# Superheroes (d)

Deze oefening is het vierde deel van de oefeningenreeks "superheroes". We gaan verder op de oplossing van het derde deel (c).

## Opgave

Werk verder op het bestand `index.html` uit oefening c. Deze pagina zou al reeds gekoppeld moeten zijn met een stylesheet `mainstyle.css`.

Voor volgende stappen uit met een editor naar keuze.

### Nieuwe pagina's aanmaken

- Maak drie nieuwe pagina's aan. Deze pagina's krijgen onderstaande namen.
    - artikel01.html
    - artikel02.html
    - artikel03.html
    - artikel04.html
- Voorzie elke pagina van een passende titel in het `head` element
- Voorzie elke pagina van onze **mainstyle.css** stylesheet
- Zorg dat je ook op elke pagina het copyright en de bron vermeld

### Inhoud overbrengen

Breng de artikels vanop onze `index` pagina in volgorde over naar hun eigen pagina. Het artikel `Over Ons` blijft op onze `index` pagina staan.

### Links toevoegen

- Voorzie een link naar de `index.html` pagina bovenaan elke pagina.
- Op de `index.html` pagina voorzie je een nieuw div element onder het `Over Ons` artikel. Daar plaats je een passende link naar de 4 nieuwe aangemaakte pagina's.


## Eindresultaat

Na je uitwerking zou je er pagina er ongeveer als volgt moeten uitzien. Hou er rekening mee dat er kleine verschillen kunnen zijn afhankelijk van de gebruikte browser. Mogelijk voegt de live server extensie van visual studio code ook een extra div toe op het einde.
